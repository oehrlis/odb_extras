# Release Notes - OraDBA Extras v0.1.2

**Release Date:** 2026-01-22

## Overview

Minor patch release fixing documentation build issues found in the MkDocs integration.
This release resolves link warnings that were preventing clean documentation builds.

## What's Fixed

### Documentation

- **Fixed MkDocs link warnings in checksums.md**
  - Removed problematic relative link `../checksumignore` that caused unrecognized link warnings
  - Fixed reference to `../scripts/build.sh` that targeted non-documentation files
  - Updated references to use inline code format instead of broken links
  - Documentation now builds cleanly without link warnings

## Files Changed

- `doc/checksums.md` - Fixed 3 problematic relative links
- `CHANGELOG.md` - Updated with v0.1.2 release information
- `VERSION` - Bumped to 0.1.2

## Testing

All tests continue to pass:

- 20 tests (11 extension structure + 9 checksum tests)
- All linting checks passing (shellcheck + markdownlint)
- Documentation builds without warnings

## Upgrade Notes

No breaking changes. This is a documentation-only release. No action required for existing installations.
