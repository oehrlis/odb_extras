# OraDBA Extras Extension v0.1.0

First stable release of the OraDBA Extras extension providing user-specific tools and utilities.

## Overview

The OraDBA Extras extension provides shell wrappers and configurations for
user-specific tools that are not available in the standard OS installation.
This release includes a comprehensive checksum management system for maintaining
file integrity when adding binary tools to your installed extension.

## What's New

### Core Features

- **Initial Extension Structure**: Complete OraDBA extension template implementation
- **User-Specific Tool Support**: Framework for adding tools like GNU tar, OCI CLI, jq, and other utilities
- **Dual Checksum Management**:
  - `bin/oradba_checksum.sh` - User-facing script for installed extensions
  - `scripts/checksum.sh` - Developer script for extension development
- **Automatic Backup**: Checksums are backed up before updates
- **Exclusion Patterns**: Supports `.checksumignore` for flexible file management

### Documentation

- **Comprehensive Documentation**: Complete guides for installation, usage, and development
- **Checksum Management Guide**: Detailed workflow for users and developers
- **Examples and Troubleshooting**: Practical examples for adding tools
- **Function Header Standards**: Standardized templates for all shell functions

### Development & Testing

- **Makefile Targets**: Complete automation for testing, linting, building, and checksums
- **CI/CD Ready**: GitHub Actions workflow for automated releases

## Installation

Download and extract the extension to your OraDBA local extensions directory:

```bash
# Download release
wget https://github.com/oehrlis/odb_extras/releases/download/v0.1.0/odb_extras-0.1.0.tar.gz

# Verify checksum
wget https://github.com/oehrlis/odb_extras/releases/download/v0.1.0/odb_extras-0.1.0.tar.gz.sha256
sha256sum -c odb_extras-0.1.0.tar.gz.sha256

# Extract to OraDBA
tar xzf odb_extras-0.1.0.tar.gz -C ${ORADBA_LOCAL_BASE}/

# Reload OraDBA environment
source ${ORADBA_BASE}/bin/oradba.sh
```

## Usage

### Adding New Tools

After installation, add your tools and update checksums:

```bash
# Navigate to extension
cd ${ORADBA_LOCAL_BASE}/odb_extras

# Add your tool (e.g., jq)
wget https://github.com/jqlang/jq/releases/download/jq-1.7/jq-linux64 -O bin/jq
chmod +x bin/jq

# Update checksums
./bin/oradba_checksum.sh

# Verify integrity
./bin/oradba_checksum.sh --verify
```

### Verifying Extension Integrity

Check if any files have been modified:

```bash
cd ${ORADBA_LOCAL_BASE}/odb_extras
./bin/oradba_checksum.sh --verify
```
